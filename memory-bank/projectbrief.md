# Project Brief: Lira Mucevherat

## Overview
Lira Mucevherat is a luxury jewelry website for a 3rd generation jeweler based in Nisantasi, Istanbul. The site features 7 pages with an ultra-luxury black + gold design aesthetic.

## Core Requirements
- 7-page website (Home, Collections, Collection Detail, Workshop, Bespoke, Store, Contact)
- Ultra-luxury visual design (black + gold palette)
- Cinematic slow animations (1-1.5s transitions)
- Dynamic [slug] routing for collection details
- Responsive design across all devices
- Turkish language content

## Target Audience
High-end jewelry clientele seeking bespoke and luxury ready-made pieces in Istanbul.

## Brand Identity
- Established: 1987
- Location: Nisantasi, Istanbul
- Heritage: 3 generations of master jewelers
- Values: Craftsmanship, tradition, timeless elegance
